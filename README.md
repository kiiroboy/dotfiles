# Installation
Clone the repository
```
git clone https://github.com/tath2701/dotfiles
cd dotfiles
```

The script installs the config files into ``$HOME/.config``. **the script will overwrite the existing files**, make sure to handle the conflicts.
```
./install.sh
```
