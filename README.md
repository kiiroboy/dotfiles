# Installation
Clone the repository
```
git clone https://github.com/tath2701/dotfiles
cd dotfiles
```

The script injects symlinks into ``$HOME/.config``. **the script will overwrite the existing files/links**, make sure to handle the conflicts.
```
./install.sh
```
